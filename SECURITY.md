# Security Policy

## Supported Versions

DIContainer Swift is available only for swift 5.1 or above.

| Version | Supported          |
| ------- | ------------------ |
| 5.1.x   | :white_check_mark: |
| 5.0.x   | :x:                |
| 4.0.x   | :x: |
| < 4.0   | :x:                |

## Reporting a Vulnerability

To report some vulnerability, please, register it in the issues section.
